Below is an instance of an {{ title }} EditorJS Materia; component. Click on the area and the block tool will appear, or you can just start typing.

The default block is a <p> paragraph, you can also add header <h1> - <h6> blocks, list with <ol> / <li> and <code> blocks.

You can also paste in an <img> url into the editor and it will start the SimpleImage plugin which provides an <img> tag pointing to the image, and with some additional controls.

When saving check the output panel below to see the data results. The Autosave feature enables the `EditorJS` instance to update the form value on a regular basis, but to save the data to a service the form would still need submission.
